Task 1 - SRAM PUF mental model and TODO

Mental model:
The FPGA SRAM/BRAM power-up state is used as a weak PUF.
Some memory cells naturally power up as 0 or 1 due to physical manufacturing variation.
The task is to read this uninitialized memory before it gets overwritten, send it to the PC over UART, and evaluate how useful it is as a PUF.

Data model:
- one line in puf_data.txt = one 256-bit PUF instance
- 512 lines = 512 PUF instances simulated from one FPGA memory space
- multiple files after power-cycles = repeated readings of the same PUF instances
- same line number across files = same PUF instance across different power-ups
