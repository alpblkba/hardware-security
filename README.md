# Hardware security labs

FPGA-based hardware security labs and experiments using Lattice iCE40HX8K, Verilog, Yosys, and nextpnr.

## Environment

- Lattice iCE40HX8K
- Verilog HDL
- Yosys
- nextpnr
- iceprog
- Python UART tooling

## Tasks

### Task 0 — FPGA bring-up and UART

Initial FPGA setup, toolchain validation, UART communication, and LED experiments.

### Task 1 — PUFs

PUF oriented hardware security experiments and analysis.
